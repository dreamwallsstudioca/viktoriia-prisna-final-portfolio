# WEBD168 Final Project — Portfolio

This folder contains a 3-page responsive portfolio website.

## Files
- `index.html` — Home page
- `portfolio.html` — Project portfolio page
- `contact.html` — Contact page
- `css/styles.css` — Custom CSS
- `images/` — Favicon, professional photo placeholder, and project images

## Important replacement
Replace `images/professional-photo.jpg` with your own professional photo.
Keep the same file name so the HTML does not need to change.
